# ACCESORIOS DM

Proyecto académico de ecommerce distribuido para la empresa **ACCESORIOS DM**, orientado a la gestión de catálogo, inventario, carrito, pagos y seguridad mediante una arquitectura basada en microservicios.

## Descripción general
El sistema fue diseñado para separar responsabilidades en distintos servicios, permitiendo escalabilidad, mantenimiento y despliegue independiente por módulos.

## Arquitectura del proyecto
La solución está compuesta por los siguientes repositorios:

- **accesorios-dm-frontend**: interfaz web en Angular
- **accesorios-dm-api-gateway**: punto de entrada único para los microservicios
- **accesorios-dm-inventory-service**: gestión de catálogo, productos, categorías y stock
- **accesorios-dm-payment-service**: base del flujo de carrito, órdenes y pagos
- **accesorios-dm-security-service**: autenticación, autorización y roles
- **PostgreSQL**: persistencia de datos

## Flujo general
Usuario → Frontend → API Gateway → Microservicio → Base de datos

- Gateway health: `http://localhost:8080/actuator/health`
